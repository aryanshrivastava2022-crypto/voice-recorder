# Voice Recorder
A simple Python application that records audio from your microphone and saves it as a WAV file.

# Features
- Record audio from microphone
- Save recordings as WAV files
- User-defined recording duration
- Easy command-line interface

# Requirements
- Python 3.x
- sounddevice
- scipy

# Installation
Install the required libraries:
pip install sounddevice scipy

# How to Run
1. Download the project files.
2. Open a terminal in the project folder.
3. Run:
voice_recorder.py
4. Enter recording duration in seconds.
5. Speak into the microphone.
6. The recording will be saved automatically.

# Sample Output
Enter recording duration (seconds): 5
🎙️ Recording...
✅ Recording saved as recorded_audio.wav
